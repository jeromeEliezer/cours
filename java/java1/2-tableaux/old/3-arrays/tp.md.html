Exercice 1

Écrivez une méthode pour renverser un string. Plutôt que d’afficher directement le résultat dans la console, vous utiliserez le mot return.

// on écrit string au début pour indiquer le type de l'objet résultat de la méthode
public static String maMethode(String minus){
  String upper = minus.toUpperCase();
  return upper;
}

// voici comment l'utiliser
public static void main (String[] args){
  String unMot = "coucou";
  String resultat = maMathode(unMot); // le res est maintenant dans la variable resultat

  System.out.println(resultat);
}

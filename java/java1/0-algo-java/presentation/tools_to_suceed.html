Bonnes habitudes

Installer les bonnes habitudes.

  • Je lis chaque jour un article en lien avec le monde du numérique.
  • Je fais tous les jours un kata (Code Wars).
  • Je révise chaque soir pendant 30 minutes les notions vues le jour même.

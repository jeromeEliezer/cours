En vous aidant des cours :

  • Java

réaliser les tâches suivantes :

1. Connexion et requêtes “Select”

  • Utiliser la base de données avion que vous avez déjà découvert pour l’apprentissage du SQL
  • Créer un nouveau projet sous Eclipse ou un autre IDE de votre choix
  • Créer une classe de connection à la base de données bd-avion
  • Créer une classe contenant les méthodes pour exécuter des requêtes suivantes (afficher les résultats dans la console) :
  • Afficher les noms de tous les pilotes
  • Afficher la liste des avions par marque (Boeing, Airbus, Caravelle, ATR)
  • Rechercher un pilote (par son nom) et afficher son nombre de vols
  • Rechercher les vols pour une ville donnée en paramètre de la méthode

2. Connexion et requêtes CRUD (Create, Read, Update, Delete)

  • Create : Ecrire une méthode qui permet d’ajouter un nouveau pilote (vous avec l’identifiant n°10 et la ville de PARIS).
  • Read : Ecrire une requête qui permet d’afficher tous les avions qui ne volent pas.
  • Update : Ecrire une méthode qui permet la mise à jour du nom d’un Pilote (Finalement, piloter ne vous interesse plus et vous changer le nom du pilote numéro 10).
  • Delete : Ecrire une méthode qui permet de détruire un ville en fonction du nom de la ville d’arrivée en paramètre (faites le pour la ville de Toulouse).
